Setup Instructions
Getting Telegram API Credentials:
  1. Go to my.telegram.org/auth
  2. Log in with your phone number
  3. Click on "API development tools"
  4. Create a new application
  5. Copy the API ID and API Hash
Setting up Groups:
  1. Create a file called groups.txt in the bot directory
  2. Add one Telegram group invite link per line
  3. Example format: https://t.me/groupname
Using the Bot:
  1. Save your configuration
  2. Go to the Dashboard
  3. Click "Connect to Telegram"
  4. Authenticate with your phone number
  5. Use the Quick Actions to control the bot
